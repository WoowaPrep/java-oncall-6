# 개발자 비상근무

---


## 기능 요구사항


### 입력 기능
- [ ] 비상 근무를 배정할 월과 시작 요일을 입력
- [ ] 평일 비상 근무 순번대로 사원 닉네임을 입력 
- [ ] 휴일 비상 근무 순번대로 사원 닉네임을 입력 (토요일, 일요일, 법정공휴일)


### 출력 기능
- [ ] 비상 근무표 출력 (**평일이면서 법정공휴일**의 경우에만 요일 뒤에 **(휴일)** 표기)


### 기능 요구사항
- [ ] 법정 공휴일
    - 1월 1일: 신정
    - 3월 1일: 삼일절
    - 5월 5일: 어린이날
    - 6월 6일: 현충일
    - 8월 15일: 광복절
    - 10월 3일: 개천절
    - 10월 9일: 한글날
    - 12월 25일: 성탄절
- [ ] 연도는 고려하지 않으며, 매년 **2월은 28일**까지만 있다고 가정
- [ ] 올바르지 않은 입력을 할 경우 **[ERROR]**로 시작하는 에러 메시지를 출력 후 다시 입력받는다.
- [ ] 비상 근무를 배정할 월과 시작 요일의 입력 값이 올바르지 않은 경우, '비상 근무를 배정할 월과 시작 요일'부터 다시 입력받는다.
- [ ] 평일 순번 또는 휴일 순번의 입력 값이 올바르지 않은 경우, '평일 순번'부터 다시 입력 받는다.
- [ ] 특정 근무자가 연속 2일 근무하게 되는 상황에는, 다음 근무자와 순서를 바꿔 편성한다. 
  - 수아가 평일인 목요일에 비상 근무를 서고, 다음 날인 금요일이 휴일이면서 순번상 또다시 수아가 근무해야 할 경우,
    다음 휴일 근무자와 순서를 바꿔서 근무한다.
  - 법정공휴일인 수요일에 수아가 비상 근무를 서고 다음 날 평일 순번이 수아인 경우에는,
    다음 평일 근무자와 순서를 바꿔서 근무한다.
  - 비상 근무자 배정 시 다음 근무자와 순서를 바꿔야 하는 경우에는, 앞의 날짜부터 순서를 변경한다. 


### 예외 처리
- [ ] 근무자 닉네임 중복인 경우
- [ ] 근무자 닉네임 5자 초과된 경우
- [ ] 근무자 수가 5명 미만인 경우
- [ ] 근무자 수가 35명 초과된 경우
